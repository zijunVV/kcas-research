# KCAS Research

KCAS Research is a knowledge-capital research system designed to identify, compare, rank, and explain long-term compounding assets.

## What is KCAS Research?

KCAS Research is NOT a stock screener.

KCAS Research is NOT a portfolio construction tool.

It is a research framework for understanding:

- What makes an asset valuable
- Why an asset compounds
- Which knowledge-capital assets deserve attention
- How assets compare against each other

Core Question:

```text
What are the highest-quality assets and why?
```

## Core Research Modes

### Deep Research Report

Analyze a single asset in depth.

Examples:

- NVIDIA
- Palantir
- Anthropic
- Stripe
- Notion
- Creator businesses

### Comparison Report

Compare multiple assets.

Examples:

- NVIDIA vs Micron
- Anthropic vs OpenAI
- Salesforce vs ServiceNow

### Ranking Report

Rank assets by:

- Asset Quality
- Knowledge Capital
- Compounding Potential
- Reflexivity
- Investment Attractiveness

### Sector Landscape Report

Map an entire sector.

Examples:

- AI Infrastructure
- Enterprise Software
- Creator Economy
- Knowledge Businesses

## Key Features

### Knowledge Capital Ranking

Manual installation:

```bash

cp -R kcas-research ~/.codex/skills/kcas-research
```

If the skill does not appear automatically:

Restart Codex.

```bash
git clone https://github.com/<your-username>/kcas-research.git
cp -R kcas-research ~/.codex/skills/kcas-research
```

## Research Philosophy

KCAS Research rewards:

- Durable moats
- Knowledge capital accumulation
- Long-term compounding
- Reflexivity
- Capital allocation quality

KCAS Research avoids:

- Short-term prediction
- Narrative chasing
- Momentum investing
- Pure valuation investing
- Unsupported speculation

## Disclaimer

KCAS Research is a research and decision-support framework.

It does not provide financial, legal, tax, or investment advice.
