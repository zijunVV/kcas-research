# KCAS Research Modes

## Shared Evidence Standard

For each asset, start from KCAS Core outputs:

- Asset Quality Score.
- Investment Attractiveness Score.
- Final Allocation Score.
- Knowledge Capital Analysis.
- Compounding Potential.
- Reflexivity Analysis.

If a KCAS Core output is missing, mark it `Missing` or `Preliminary` and state what evidence is required.

Use these labels:

- Facts: observed evidence.
- Interpretations: reasoned conclusions from evidence.
- Assumptions: claims that must become true for the thesis to work.
